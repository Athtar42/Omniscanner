# Omniscanner
Network forensic tool

## Requirements
Need WinPcap(or Npcap) and TShark installed 

For windows please install Tshark(Wireshark) using choco 

`choco install wireshark `

## Packages needed
GeoIP2 pyshark scapy 

## Install
`$ pip install geoip2  `   
`$ pip install scapy  `   
`$ pip install pyshark  `
